PoisedProject is a project tracking program for an construction company called Poised.
there are currently two classes in it which are ACC and Project. ACC keeps track 
of the Architect Contractor and Client or Customer by taking in thier details.
The Prject class keeps track of the projects the amount paid the, the total cost of the project,
the project name, project ERF number, the physical address of the project, the building type and the dead line

the main class PosiesdProject collects all the data collected and prints it on the screen. Also shows a menu
on which the user can add data like the if the customer paid moeny for the building or house or to create a  invoice or to
finalize the project.
