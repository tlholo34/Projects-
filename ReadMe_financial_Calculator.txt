Financial Calculator asks the user to choose either Investment or Bond.

if the user chose Investment the program calculates the the investment
by asking the user questions which are needed to calculate the return using the 
compound interst formula or simple interst formula depending on what the users entered 

if the user chose Bond the the program would ask the user questions needed for the formula that calculates
the amount the user has to pay and the amount of months the uses has to pay for to settle the bond on the interest 
the user chose
